# 特性
---

- 服务器端口检测
- 重复大文件检测
- QQ截屏
- Windows 7登录壁纸修改
- 多文件管理器
- 二维码识别（本地图片，摄像头，网页）
- 快速打开Hosts文件

# 源码构建
---
```
git clone git@github.com:qmjy/DevKits.git
cd DevKits
mvn clean package assembly:single
```
